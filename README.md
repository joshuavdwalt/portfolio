## 🚀 How to Run Locally

1. **Clone the repository:**
   ```bash
   git clone https://github.com/joshuavdwalt/portfolio.git
   ```

2. **Navigate into the project folder:**
   ```bash
   cd portfolio
   ```

3. **Open in your browser:**
   - Simply open `index.html` in any modern web browser (Chrome, Firefox, Edge, Safari).
   - No server or installation required — it runs as a static site.

## 🌍 Deployment — GitHub Pages

This site is hosted for free using **GitHub Pages**. To deploy your own version:

1. Push the code to a GitHub repository named `portfolio` (or any name).
2. Go to **Settings → Pages**.
3. Under **Branch**, select `main` and click **Save**.
4. Your site will be live at `https://<your-username>.github.io/<repo-name>/`.
