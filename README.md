# Joshua van der Walt — Portfolio Website

A personal portfolio website built with HTML, CSS, and JavaScript as part of the **CMPG323 Profile Development** assignment at **North-West University (NWU)**.

## Live Site

[https://joshuavdwalt.github.io/portfolio/](https://joshuavdwalt.github.io/portfolio/)

## About

This site serves as my Virtual CV and includes the following sections:

- Home
- About Me
- Skills
- Projects
- Contact

## Technologies Used

- HTML5
- CSS3 (with animations)
- JavaScript

## How to Run Locally

1. Clone the repo: `git clone https://github.com/joshuavdwalt/portfolio.git`
2. Open `index.html` in your browser
